Bootstrap „Block Grid” with FreestyleGrid

Similar concept like Block Grid in Foundation. Done with the kind help of FreestyleGrid.

Here is an example in which we have this grid: 2 columns on extra small (xs), 3 columns on small (sm), 5 columns on medium (md) and 7 columns on large (lg) displays.
Try to change width of the window. 

<ul class="fgrid fgrid-xs-2 fgrid-sm-3 fgrid-md-5 fgrid-lg-7">
  <li class="fgrid-item">…</li>
  …
</ul>

Are you curious about the source code? Check it out here.
